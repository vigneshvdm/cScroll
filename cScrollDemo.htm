About Us
cScroll is a very light weight plugin that is developed to serve the purpose of a scroll bar.
At the same time it is light weight which will improve the performance of application since it takes very less time to load
Anyway we would recommend you to copy the contents of the plugin and have it in one of the js file you use to reduce the number of server hits
